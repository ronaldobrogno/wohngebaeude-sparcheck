# Wohngebäude-SparCheck (GitHub Pages)

Statische SEO-Landingpage + Funnel (HTML/CSS/JS) für Lead-Generierung Wohngebäudeversicherung.

## Setup
1. Repo auf GitHub erstellen, Dateien hochladen.
2. GitHub Pages aktivieren (Settings → Pages → Deploy from branch).
3. In `index.html` den Form-Endpunkt setzen:
   - Formspree: action="https://formspree.io/f/DEINCODE"
   - oder Netlify Forms (Form-Attribute anpassen)
